**Automated Parasite Assignment**

A new dropdown on the Shipyard tab of the Economics window allows you to select a fleet as the source of parasites for that population

When a new ship is built, if it has a specified strikegroup listed, it will automatically create a squadron called Strikegroup, then assign and land parasites from the parasite source fleet. If the entire specified strikegroup is not available, it will land a partial strikegroup based on the ships available in the source fleet.
