**Tactical Map Events**

Tactical Map Events are available in v1.9. Click the Events checkbox to toggle.

**Order Delay**

Order delay is now implemented. It is working with cycle orders, repeat orders and move templates.

**Conventional Reactor**

Added a conventional reactor with 1/4 the power output of the pressurised water reactor.

**Civilian Ground Forces Toggle**

A new Show Civilian checkbox on the Formation Templates tab of the Ground Forces window allows you to toggle civilian templates and unit classes

**Fighter Construction Event**

A new Fighter Construction event with no interrupt replaces the current ship construction event for fighters

**New Events**

Event added for starting a queued research project.

Event added for starting a queued industrial project.

Event added for deployment time exceeded.

**Low G Habitable Worlds**

Add checkbox to separate out display of low G habitable worlds.


**Distance Measuring**

To measure distances on the Tactical Map, hold down shift and drag a mouse. A line will be displayed, along with the length of the line in kilometres.

While shift is held down, the start of the line will remain static, while the end of the line can be dragged around.


**Multiple Window Option**

Currently, a new window will open every time you click the appropriate button.

For v1.9, clicking the button will only open a window if none exist. Otherwise, it will bring the current window to the front.

If you shift-click the button, it will open a new window regardless of how many are currently open.

The following windows may only have ever have one instance open:
Create Project
Missile Design
Turret Design

The Medals Window is a dialog window so retains focus while it is open.
